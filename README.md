## How to install qemu

    sudo apt-get update
    sudo apt-get install qemu-system-arm

## How to install xargo

    cargo install xargo

## How to install nightly

    rustup install nightly

## Use UART

    e.g.) picocom -b 115200 /dev/pts/28
